# 테트리스 (학습용)

HTML, CSS, JavaScript만 사용하는 브라우저 테트리스 프로젝트입니다.

## 실행 방법

1. 이 폴더를 에디터(Cursor, VS Code 등)에서 엽니다.
2. `index.html` 파일을 브라우저로 엽니다.
   - 파일 탐색기에서 `index.html`을 더블클릭하거나
   - 브라우저 주소창에 파일 경로를 입력합니다.
3. **시작** 버튼을 누르거나 **재시작** 버튼으로 게임을 시작합니다.
4. 게임 보드에 블록이 떨어지고, 점수·다음 블록·홀드 패널이 보이면 성공입니다.

빌드나 로컬 서버 없이 정적 파일만으로 실행됩니다.

## 조작법

| 키 | 동작 |
|----|------|
| `←` `→` | 좌우 이동 |
| `↓` | 소프트 드롭 (한 칸 내리기, 점수 +1) |
| `↑` | 회전 (SRS 벽 킥) |
| `Space` | 하드 드롭 (즉시 착지, 칸당 점수 +2) |
| `C` 또는 `Shift` | 홀드 (블록 보관·교환, 블록당 1회) |
| `P` | 일시정지 / 재개 |

- **시작**: 새 게임 시작
- **재시작**: 점수·보드·레벨을 초기화하고 새 게임 시작

## 레벨 시스템

| 항목 | 설명 |
|------|------|
| 시작 레벨 | 1 |
| 레벨 업 조건 | 누적 점수가 `현재 레벨 × 500`점에 도달 |
| 낙하 속도 | 레벨 1: 500ms → 레벨마다 35ms 감소 (레벨 13부터 최소 80ms 고정) |
| 최고 레벨 | 20 (누적 10,000점 도달 시) |

예: 레벨 1 → 500점, 레벨 2 → 1000점, 레벨 3 → 1500점 달성 시 레벨 업

## 기능 목록

- 7가지 테트로미노(I, O, T, S, Z, J, L) 7-bag 랜덤 생성
- 자동 낙하 및 좌우 이동
- SRS 벽 킥 회전
- 소프트 드롭 (↓ 키, 한 칸당 점수 +1)
- 하드 드롭 (Space, 칸당 점수 +2)
- 홀드 (C / Shift, 블록 고정 전 1회)
- 다음 블록 미리보기
- 벽·바닥·고정 블록 충돌 판정
- 블록 고정 및 라인 클리어
- 1~4줄 점수 (100 / 300 / 500 / 800)
- 레벨 시스템 (레벨별 목표 점수 달성 시 레벨 업, 낙하 속도 증가)
- 게임 오버 (생성 위치 막힘)
- 일시정지
- 재시작
- 모바일 터치 조작

## 파일 구성

| 파일 | 설명 |
|------|------|
| `index.html` | 화면 구조 (보드, 점수·레벨, 홀드/다음, 버튼, 조작법) |
| `style.css` | 레이아웃과 스타일 |

| `docs/dev-log.md` | 개발 진행 로그 |
